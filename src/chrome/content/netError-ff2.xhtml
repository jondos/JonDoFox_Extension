Haben Sie das Programm JonDo gestartet?

Bitte starten sie das JonDo Programm. Anderfalls können Sie keine Verbindung zum JonDonym-Dienst herstellen.

Bitte installieren und starten Sie das Programm JonDo.

Bitte deaktivieren Sie vorübergehend die JonDo Schnittstelle durch wählen von "None" im Proxy Menü.

Klicken Sie nun hier um die JonDo Webseite zu besuchen und das JonDo Programm herunterzuladen.

Nachdem Sie das JonDo Programm gestartet haben setzen Sie die Proxy Einstellungen zurück zu "JonDo".





ACHTUNG: Falls Sie die Einstellungen nicht zu "JonDo" zurücksetzen surfen Sie ohne Anonymitätsschutz durch das Internet!

Abschließender Test

Klicken Sie nun hier um Ihre Installation mit dem JonDonym Anonymitätstest zu überprüfen.

Falls eines der Felder im Anonymitätstest rot ist kann man Sie im Internet identifizieren oder wiedererkennen. In diesem Fall sollten Sie ihre Einstellungen überprüfen und die zugehörige Hilfe lesen.
